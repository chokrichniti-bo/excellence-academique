⚙️ Configuration requise

Cette version en ligne nécessite Firebase.

1. Allez sur console.firebase.google.com et créez un projet gratuit.
2. Ajoutez une application Web (icône </>) et copiez le bloc firebaseConfig.
3. La configuration est déjà insérée dans ce fichier.
4. Dans Firebase : Firestore Database → Créer une base de données → mode test.

Ensuite, tout fonctionnera pour tous vos étudiants, sur tous leurs appareils.
